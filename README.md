# AppArte

_Aplicación web destinada al registro de eventos_


### Instalación 🔧

_Ejecutar_

```
Composer installl
```

_Generar la clave del proyecto_

```
php artisan key:generate
```

_Configurar en archivo .env_

_Ejecutar las migraciones_

```
php artisan migrate
```

_Arrancar el servidor_

```
php artisan serve
```
